C Programming Fundamentals

C++ & Modern Systems Programming

Level up to modern C++ for embedded UIs, complex system software, and clean object-oriented design using C++17/20 standards.

OOP & RAII Templates & STL Smart Pointers Move Semantics CMake Qt Framework C++17 / C++20

POSIX APIs & Concurrency

View Guide

Write Linux system software using the POSIX standard — processes, inter-process communication, UNIX sockets, and multithreading.

POSIX File I/O Processes & fork() Signals UNIX Sockets Pthreads Mutexes & Semaphores Pipes & Shared Memory
Phase 03
Embedded Linux Developer
Cross-Compilation, Bootloaders & the Kernel
Stage 05

Cross-Compilation & Toolchains

View Guide

Build code for a target architecture from your host machine. Master the GNU toolchain, understand sysroots, and emulate hardware with QEMU before touching real boards.

GNU Toolchain Sysroot & Libraries Static vs Dynamic Linking QEMU Emulation readelf / objdump / nm Cross-Compile with CMake
Stage 06

Bootloaders (U-Boot)

View Guide

Understand how a system comes alive from power-on. Master U-Boot configuration, board porting, and Device Tree — the glue between hardware and the kernel.

Boot Sequence U-Boot Config & Build U-Boot Commands & Scripting Device Tree (DTS) Board Porting TFTP & Network Boot
Stage 07

Linux Kernel

View Guide

Dive into the kernel — configure it for your hardware, build it, understand its subsystems, and learn to debug it in the field.

Kconfig & menuconfig Kernel Compilation Boot Parameters Kernel Subsystems printk & Dynamic Debug ftrace & kprobes
Phase 04
Kernel Developer
Device Drivers, Root Filesystem & Build Systems
Stage 08

Device Drivers & Kernel Modules

Write code that runs inside the kernel. Build character drivers, platform drivers, and interface with real hardware peripherals over I2C, SPI, and UART.

Module Lifecycle (init/exit) Character Devices Platform Drivers Interrupts & Timers I2C / SPI / UART sysfs & procfs DMA
Stage 09

Root Filesystem & Init Systems

Build the environment your applications will run in. Understand file system types, init systems, and how to create minimal rootfs images for production.

BusyBox systemd SysVinit ext4 / squashfs / tmpfs / overlayfs NFS for Development initramfs
Stage 10

Build Systems — Yocto & Buildroot

Automate the entire toolchain, kernel, rootfs, and package build. Learn Buildroot for simplicity and Yocto/OpenEmbedded for production-grade flexibility.

Buildroot Basics Buildroot Packages & Board Config BitBake & Recipes Yocto Layers Custom Distro SDK Generation
Phase 05
Production Engineer
Debugging, Profiling & Advanced Topics
Stage 11

Debugging & Profiling

Find and fix problems in kernel space and user space. Use GDB cross-debugging, trace system calls, profile CPU and memory, and debug with JTAG on real hardware.

GDB + gdbserver strace / ltrace perf Valgrind / ASan JTAG / OpenOCD crash & kdump
Stage 12

Advanced System Topics

Production-ready systems require real-time guarantees, security hardening, and robust update mechanisms. These topics separate engineers from experts.

PREEMPT_RT Latency Analysis SELinux / AppArmor Secure Boot OTA Updates (SWUpdate / RAUC) Power Management Containers on Embedded
