Embedded Linux Reference

Essential commands, concepts, and workflows — everything you reach for daily as an embedded Linux engineer.

Boot Sequence
ROM / BootROM Runs from mask ROM, initialises minimal hardware
SPL / MLO Secondary Program Loader — fits in SRAM, loads U-Boot
U-Boot Full bootloader — sets up DDR, loads kernel + DTB
Linux Kernel Decompresses, probes hardware via DT, mounts rootfs
Init (systemd) PID 1 — starts user-space services
